Blog De Vacare

Le Blog de Vacare est un projet Next.js servant un besoin simple : servir de site où je pourrais rédiger des blogs avec des personnages fictifs et exporter ces derniers sur une page web statique hébergée sous Github Pages.

Stack Technique

  • Frontend : Next.js 15, React, TypeScript, TailwindCSS
  • Backend : Node.js 15, MySQL, API REST, Tests unitaires vitest
  • Architecture : Monorepo, séparation des responsabilités par couche, Injecteur de Dépendances
  • Déploiement : GitHub Pages (export statique), XAMPP/Apache (hébergement local développement)
  • Sécurité : JWT avec refresh tokens, middleware d'authentification, hachage bcrypt de mots de passe

Blog multi-utilisateurs !

Le Blog de Vacare consiste principalement en une application où différents utilisateurs peuvent envoyer des messages et se répondre entre eux dans des blogs du site.

Chaque blog est contenu dans un dossier, permettant de les trier par catégorie et de facilement retrouver des données recherchées !

Monorepo Next.js multi-projets

Bien que ce projet soit mon premier sous Next.js, j'ai décidé de vite expérimenter avec plusieurs projets à la fois, notamment afin de pouvoir gérer les données du site dans un projet Backend et l'affichage à travers un projet frontend, tout en gardant un projet partagé pour les éléments partagés entre les 2 projets

Une telle structure me permet aujourd'hui de démarrer les 2 projets à la fois et de rapidement les débuguer, et me permettra à terme d'ajouter de nouveaux projets à ce monorepo qui enrichiront tout ce que ce site a à raconter !

Première découverte du site

En arrivant sur le site, vous devriez faire face à la page d'accueil jointe. Le site se compose actuellement de 2 modes d'affichage : Un affichage rétro (disponible sous le domaine "/old") et un affichage moderne, les 2 ayant accès à la même base de données mais affichant les éléments différemment, comme il sera décrit plus bas.
